Inline CSS

This is the CSS which is applied inside the HTML documents . It is used with attributes

Internal CSS

This is the CSS which is on the same webpage with the HTML document. It is uses a style tag.

External CSS

This is the CSS which is applied by creating a css file and linking it to an HTML document.

This is best used when having many web pages to style

Last Rule

- The first formarts of lets say our paragraph on top has the syles and we apply a different styles value targeting the same paragraph, the last styles will apply

Specificity

- Inline CSS overides all the CSS styles, it is more specific

- Class (.) selectors has more specificity than the element selectors.

- ID (#) selectors has more specificity than the class selectors

Univeral Selectors (*)

Selects all the elements and it has the least amount of specificity.
