Inline CSS

This is the CSS which is applied inside the HTML documents. It is used with style

Internal CSS

This is the CSS which is on the same webpage with the HTML document. It is uses a style tag.

External CSS

This is the CSS which is applied by creating a css file and linking it to an HTML document.
